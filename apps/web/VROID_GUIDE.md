# 🎀 VRoidモデルの使い方ガイド

## 1. VRoid Studioでモデルを作成

1. **VRoid Studioをダウンロード**
   - https://vroid.com/studio
   - 無料で使えます！

2. **可愛いモデルを作成**
   - 髪: 翠青色 (#1E6F68)
   - 目: 大きく、キラキラ
   - 服: 白いワンピース
   - 雪の結晶や星のアクセサリー

3. **VRM形式でエクスポート**
   - 「撮影・エクスポート」→「エクスポート」
   - VRM形式を選択
   - `hakusan-avatar.vrm` として保存

## 2. モデルをプロジェクトに配置

モデルファイルを以下の場所に配置:
```
apps/web/public/models/hakusan-avatar.vrm
```

## 3. コードで読み込む

`apps/web/src/avatar/VRoidAvatar.ts` を作成済み！
自動的にVRoidモデルを読み込みます。

## 4. 確認

http://localhost:5173 にアクセスして確認！

---

## 💡 Tips

- VRoidモデルの方が圧倒的に可愛い！
- 表情やポーズも自由自在
- OpenSeeFaceで顔トラッキングも対応
- 商用利用も可能（モデルのライセンス次第）

## 🎨 推奨カラー

- **髪**: #1E6F68 (翠青)
- **肌**: #FFD5B5 (温かみのある肌色)
- **目**: #1E6F68 (翠青、髪と同じ)
- **服**: #F7F7F7 (雪白) / #F0F8FF (淡い青)
- **アクセント**: #FFD700 (金色の星)
